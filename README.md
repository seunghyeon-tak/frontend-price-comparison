# 🛒 상품 가격 비교 웹 서비스 (Frontend)

Vue 3 + Vite + TailwindCSS 기반의 프론트엔드 프로젝트입니다.

---

## 🚀 프로젝트 소개

- 상품 실시간 가격 비교를 위한 프론트엔드
- 로그인 / 회원가입 UI
- 카카오 로그인 버튼 포함
- Tailwind CSS로 반응형 지원

---

## 🧑‍💻 기술 스택

- **Vue 3**
- **Vite**
- **TypeScript**
- **Tailwind CSS (v3.4.3)**

---

## 🛠️ 설치 및 실행 방법

```bash
# 프로젝트 클론
git clone https://github.com/your-username/front-price-comparison.git
cd front-price-comparison

# 패키지 설치
npm install

# 개발 서버 실행
npm run dev
